# Restaurant_SQL_Project
SQL project: Restaurant database with 10 tables, analytical queries, and reporting views
**Description:**  
This project demonstrates a fully designed SQL database for a restaurant, including 10 tables, data insertion scripts, analytical queries, and reporting views.

**Database Features:**
- Customer management (`Customers` table)  
- Employee management (`Employees` table)  
- Table reservations (`Tables` & `TableReservations`)  
- Menu items and ingredients (`MenuItems`, `Ingredients`, `MenuIngredients`)  
- Orders and order details (`Orders`, `OrderDetails`)  
- Suppliers management (`Suppliers` table)

**Scripts Included:**
- `RestaurantDB_Script.sql` → Database creation and data insertion  
- `Views_Script.sql` → Views for order summaries, popular menu items, customer reservations, and full reports

**Outputs:**
- `Outputs/` folder contains CSV files and screenshots of query results for documentation and demonstration purposes

**Usage:**
1. Run `RestaurantDB_Script.sql` to create the database and insert sample data  
2. Run `Views_Script.sql` to create the reporting views  
3. Explore the outputs in the `Outputs/` folder for query results
